What is shelf_asset_router?

shelf_asset_router is a Flutter package that provides multi-route asset handling for Shelf HTTP servers.

It allows you to serve static files from Flutter's bundled assets, making it perfect for integrating web content into native Flutter applications via WebView.

How It Works

  1. Configure routes with AssetRoute
  2. Create a handler with AssetHandler.create()
  3. Start a Shelf server on localhost
  4. Load the content in a WebView

Example Code

final handler = AssetHandler.create([
  AssetRoute(
    basePath: 'webapp',
    defaultDocument: 'index.html',
  ),
]);

final server = await shelf_io.serve(
  handler,
  'localhost',
  8080,
);
